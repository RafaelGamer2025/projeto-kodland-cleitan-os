# Cleitan-OS 🚀

O **Cleitan-OS** é uma ferramenta multifuncional desenvolvida em Python para análise de sistemas, segurança e automação. O projeto utiliza Inteligência Artificial (Google Gemini) para processamento de dados e oferece módulos especializados para varredura de rede e utilitários de sistema.

## 🛠️ Funcionalidades

- [cite_start]**Módulo de IA (`core/ai.py`):** Integração com o Google Generative AI para análises inteligentes[cite: 80, 82].
- [cite_start]**Scanner de Portas (`core/ports.py`):** Ferramenta para verificação de portas abertas e serviços[cite: 80, 83].
- [cite_start]**Scanner de Rede/Web (`core/scanner.py`):** Varredura estruturada para busca de informações e padrões[cite: 81, 84].
- [cite_start]**System Tools (`core/system_tools.py`):** Utilitários de baixo nível para interação com o sistema operacional[cite: 81].
- [cite_start]**Interface Intuitiva (`ui/interface.py`):** Interface de linha de comando (CLI) organizada para facilitar o uso[cite: 86, 88].
